# Employee Management System

This is a full-stack MERN (MongoDB, Express, React, Node.js) web application to manage employees with authentication.

## ✅ Features

- Add new employee with name, email, position, salary, and joining date
- View employee list
- Edit employee details
- Delete employee
- Login system with JWT authentication

## 🚀 Technologies Used

### Frontend

- React.js
- Axios
- React Router

### Backend

- Node.js
- Express.js
- MongoDB (with Mongoose)
- JSON Web Token (JWT) for authentication

## 💻 Installation & Setup

1. Clone this repo:git clone https://github.com/rajeshravi2005/employee-management-system.git

2. Install dependencies:

**Backend:**

cd backend
npm install
npm start

**Frontend:**

cd frontend
npm install
npm start

3. Make sure MongoDB is running.

## 📁 Folder Structure

- `/frontend` - React frontend
- `/backend` - Express backend with MongoDB
- `.env` - environment variables (not pushed to GitHub)
